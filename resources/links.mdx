## Installation Methods

Multiple tools and platforms support the Agent Skills standard. Choose the method that works best for your workflow.

<CardGroup cols={2}>
  <Card title="skills CLI" icon="download" href="https://skills.sh/">
    Install skills directly with `npx skills add`
  </Card>
  
  <Card title="Playbooks" icon="book" href="https://playbooks.com/skills">
    Install via Playbooks: `npx playbooks add skill`
  </Card>
  
  <Card title="Context7" icon="code" href="https://context7.com/docs/skills">
    Install via Context7: `npx ctx7 skills install`
  </Card>
  
  <Card title="Gumloop" icon="robot" href="https://docs.gumloop.com/core-concepts/skills">
    Upload skills directly to Gumloop agents
  </Card>
</CardGroup>

<Note>
All of these tools support the same open Agent Skills standard, so skills work consistently across platforms.
</Note>

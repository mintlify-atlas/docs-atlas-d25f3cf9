## Project Background

Webflow Agent Skills is an open-source collection of specialized skills designed to enhance the capabilities of AI agents when working with Webflow. These skills provide agents with:

- **Specialized instructions** for Webflow-specific tasks
- **Reference documentation** for Webflow APIs and best practices
- **Helper scripts** to streamline common workflows
- **Static assets** and templates for rapid development

Each skill is carefully crafted to follow the [open Agent Skills standard](https://agentskills.io), ensuring compatibility across any AI agent that supports this specification.

### Why This Matters

Agent Skills are folders of instructions, scripts, and resources that agents can discover and use to perform tasks more accurately and efficiently. By providing context-specific knowledge about Webflow's ecosystem, these skills enable AI agents to:

- Generate more accurate code for Webflow integrations
- Understand the nuances of different Webflow APIs
- Follow best practices and avoid common pitfalls
- Work faster with pre-configured templates and workflows

<Note>
Webflow Agent Skills work across any AI agent that supports the open Agent Skills standard, including Claude, Cursor, Gumloop, and more.
</Note>

# Installation

Webflow Agent Skills can be installed through several methods depending on your preferred AI agent platform or workflow.

## Skills CLI (Recommended)

The fastest way to install skills is using the [skills](https://skills.sh/) CLI:

<Steps>
  <Step title="Install all skills">
    Install the complete collection of Webflow Agent Skills:
    
    ```bash
    npx skills add 224-industries/webflow-skills
    ```
  </Step>
  
  <Step title="Install specific skills (optional)">
    If you only need specific skills, install them individually:
    
    ```bash
    npx skills add 224-industries/webflow-skills --skill webflow-designer-api
    ```
  </Step>
  
  <Step title="List available skills (optional)">
    View all available skills before installing:
    
    ```bash
    npx skills add 224-industries/webflow-skills --list
    ```
  </Step>
</Steps>

## Installation Methods

<CodeGroup>

```bash Skills CLI
# Install all skills
npx skills add 224-industries/webflow-skills

# Install specific skills
npx skills add 224-industries/webflow-skills --skill webflow-designer-api

# List available skills
npx skills add 224-industries/webflow-skills --list
```

```bash Claude Code / Factory Droid Plugin
# Add the plugin (includes all skills)
/plugin add 224-industries/webflow-skills
```

```bash Playbooks
# Install via Playbooks
npx playbooks add skill 224-industries/webflow-skills
```

```bash Context7
# Install via Context7
npx ctx7 skills install /224-industries/webflow-skills
```

</CodeGroup>

## Manual Download

You can also download `.skill` files directly and upload them to any agent that supports the [Agent Skills standard](https://agentskills.io).

<Steps>
  <Step title="Download skill files">
    Download the `.skill` file for your desired skill:
    
    | Skill | Description | Download |
    | ----- | ----------- | -------- |
    | webflow-browser-api | Control Webflow Analyze and Optimize from JavaScript | [Download](https://skills.224ai.au/webflow-browser-api.skill) |
    | webflow-code-components | Build React code components for Webflow via DevLink | [Download](https://skills.224ai.au/webflow-code-components.skill) |
    | webflow-designer-api | Build Designer Extensions or use the Designer API Playground | [Download](https://skills.224ai.au/webflow-designer-api.skill) |
    | webflow-enterprise-api | Access Enterprise-only Webflow API endpoints | [Download](https://skills.224ai.au/webflow-enterprise-api.skill) |
    | webflow-webhooks | Receive and verify Webflow webhooks | [Download](https://skills.224ai.au/webflow-webhooks.skill) |
  </Step>
  
  <Step title="Upload to your agent">
    **Claude Web / Desktop / Cowork:**
    
    1. Go to **Customize** > **Skills**
    2. Click **+**
    3. Click **Upload a skill**
    4. Select the downloaded `.skill` file
  </Step>
</Steps>

## Gumloop Agents

For [Gumloop](https://docs.gumloop.com/core-concepts/skills) users:

<Steps>
  <Step title="Download the skill file">
    Download the `.skill` file from the table above
  </Step>
  
  <Step title="Open agent configuration">
    In Gumloop, open your agent's configuration and scroll to the **Skills** section
  </Step>
  
  <Step title="Upload the skill">
    Click **+ Skill** and upload the downloaded `.skill` file
  </Step>
</Steps>

## Verify Installation

After installation, verify that the skills are available to your agent by asking it to list installed skills or checking your agent's skills configuration.

<Tip>
  Start with a single skill like `webflow-designer-api` to test your installation before adding the complete collection.
</Tip>

## What are Agent Skills?

Agent Skills are folders of instructions, scripts, and resources that agents can discover and use to do things more accurately and efficiently. They work across any AI agent that supports the [open Agent Skills standard](https://agentskills.io).

## How Agent Skills Work

Each skill contains:

- **SKILL.md** - Entry point with overview and reference index
- **references/** - Detailed API guides and documentation
- **scripts/** - Helper scripts for the skill
- **assets/** - Static assets like CSS and images

When you install a skill, AI agents can discover and use these resources to work more accurately with Webflow, providing:

1. **Specialized instructions** for Webflow-specific tasks
2. **API references** for accurate implementation
3. **Code examples** and patterns
4. **Helper scripts** to automate common workflows
5. **Best practices** from Webflow professionals
